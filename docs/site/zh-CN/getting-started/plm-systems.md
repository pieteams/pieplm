# PLM 产品生命周期管理系统清单

> 本文档整理 **产品生命周期管理（PLM, Product Lifecycle Management）** 的主流产品，覆盖：
>
> * 🌍 国际厂商 / 🇨🇳 中国厂商
> * 💰 商业收费 / 🔓 开源（含社区版）
> * 🏭 离散制造 / 流程制造 / 通用型 PLM

> 参考来源：Gartner Magic Quadrant for PLM、CIMdata、GitHub、各厂商官网公开信息（截至 2025）

适用于 **PLM 平台选型、技术对标、方案设计、RFP 编制** 等场景。

## 三、开源 / 社区版 PLM 系统

> ⚠️ 多数开源 PLM 偏轻量级，企业级使用需二次开发。

### 1. DocDoku PLM ⭐ 281

* **GitHub**：[https://github.com/docdoku/docdoku-plm](https://github.com/docdoku/docdoku-plm)
* **授权**：AGPL-3.0
* **类型**：企业级开源 PLM
* **技术栈**：Java / Maven / Docker
* **特点**：
  * 文档管理（版本控制、工作流、模板）
  * 产品结构管理（搜索、过滤）
  * BOM 管理（原材料清单）
  * 变更管理（跟踪修改）
  * WebGL 3D 可视化
  * 支持 Word、PDF、CAD 文档预览
* **适用场景**：需要完整 PLM 功能的团队

---

### 2. GitPLM ⭐ 113

* **GitHub**：[https://github.com/git-plm/gitplm](https://github.com/git-plm/gitplm)
* **授权**：Apache-2.0
* **类型**：Git 原生 PLM
* **技术栈**：Go
* **特点**：
  * BOM 管理（源 BOM + 零部件数据库）
  * 自动化发布流程
  * 零部件编号系统（CCC-NNN-VVVV）
  * CSV 零部件数据库
  * TUI 终端界面
  * KiCad 集成
* **适用场景**：小型硬件团队、KiCad 用户

---

### 3. iBizPLM ⭐ 105

* **GitHub**：[https://github.com/iBizLab/plm](https://github.com/iBizLab/plm)
* **授权**：MIT
* **类型**：研发管理平台
* **技术栈**：Spring Boot + Vue 3
* **特点**：
  * 产品管理、项目管理、测试管理
  * 知识管理、效率度量
  * AI 智能协作（6 个 AI Agent）
  * 高度可定制
  * Docker Compose 部署
* **适用场景**：研发团队、敏捷开发

---

### 4. Part-DB ⭐ 166

* **GitHub**：[https://github.com/Part-DB/Part-DB-legacy](https://github.com/Part-DB/Part-DB-legacy)
* **授权**：GPL-2.0
* **类型**：电子元器件数据库
* **技术栈**：PHP + MySQL
* **特点**：
  * 存储位置、封装、分类管理
  * 供应商、数据手册、价格管理
  * 条形码生成与扫描
  * REST API
  * 多用户权限系统
* **适用场景**：电子工程师、元器件管理

---

### 5. BluePLM ⭐ 20

* **GitHub**：[https://github.com/bluerobotics/bluePLM](https://github.com/bluerobotics/bluePLM)
* **授权**：MIT
* **类型**：轻量级 PLM（桌面应用）
