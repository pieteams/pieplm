# 国际 PLM 咨询、实施与软件厂商清单

本文档整理 **国际市场常用的产品生命周期管理（PLM, Product Lifecycle Management）相关厂商**，涵盖 **咨询公司、平台厂商、实施服务商**。清单以国际主流选型视角为主，同时 **包含在国际项目中常见的中国公司**。

> 参考来源：CIMdata PLM Solution Provider Directory、Gartner Magic Quadrant、各厂商官网公开信息（截至 2025）

> 适用对象：
>
> * 跨国企业 / 出海企业 PLM 选型
> * 全球 PLM 项目（从概念到退市）
> * 对标国际厂商能力，做方案或竞品分析

---

## 一、厂商分类说明

国际 PLM 项目通常按以下角色组合推进：

* **A 类：国际咨询公司（战略 + 治理）**
  提供全球 PLM 战略、流程设计、组织变革管理。

* **B 类：国际 PLM 平台厂商**
  提供成熟的 PLM 软件产品，是项目技术核心。

* **C 类：国际实施服务商 / SI**
  负责平台实施、系统集成、本地化交付。

> 常见组合：**A 类咨询 + B 类平台 + C 类实施（或同一厂商多角色）**。
