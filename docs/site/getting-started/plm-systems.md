## 3. Open Source / Community Edition PLM Systems

> ⚠️ Most open source PLM systems are lightweight — enterprise use requires secondary development.

### 1. DocDoku PLM ⭐ 281

* **GitHub**: [https://github.com/docdoku/docdoku-plm](https://github.com/docdoku/docdoku-plm)
* **License**: AGPL-3.0
* **Type**: Enterprise-grade open source PLM
* **Tech Stack**: Java / Maven / Docker
* **Features**:
  * Document management (version control, workflows, templates)
  * Product structure management (search, filtering)
  * BOM management (raw materials list)
  * Change management (track modifications)
  * WebGL 3D visualization
  * Supports Word, PDF, CAD document preview
* **Typical Scenarios**: Teams needing complete PLM functionality

---

### 2. GitPLM ⭐ 113

* **GitHub**: [https://github.com/git-plm/gitplm](https://github.com/git-plm/gitplm)
* **License**: Apache-2.0
* **Type**: Git-native PLM
* **Tech Stack**: Go
* **Features**:
  * BOM management (source BOM + parts database)
  * Automated release process
  * Part numbering system (CCC-NNN-VVVV)
  * CSV parts database
  * TUI terminal interface
  * KiCad integration
* **Typical Scenarios**: Small hardware teams, KiCad users

---

### 3. iBizPLM ⭐ 105
